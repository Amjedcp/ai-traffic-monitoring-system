# 🚦 AI Traffic Monitoring System using YOLOv8

## 📌 Overview

This project is an advanced AI-based traffic monitoring system that detects, tracks, and analyzes vehicles in real time.

## 🚀 Features

* Vehicle detection (car, bus, truck, motorbike)
* Object tracking using ByteTrack
* Speed calculation system
* Overspeed alert detection 🚨
* Direction-based counting (UP / DOWN)
* CSV logging of traffic data

## 🛠️ Tech Stack

* Python
* OpenCV
* YOLOv8 (Ultralytics)
* NumPy

## 📊 Output

* Processed video with bounding boxes
* CSV file with traffic data

## ▶️ How to Run

1. Open notebook in Google Colab
2. Run all cells
